# お問い合わせフォーム セットアップガイド

このガイドでは、お問い合わせフォームの送受信機能を有効化する手順を説明します。

## 🚀 Formspreeを使用したセットアップ（推奨）

### 1. Formspreeアカウントの作成

1. [Formspree](https://formspree.io/)にアクセス
2. 「Get Started」をクリックして無料アカウントを作成
3. メールアドレスとパスワードを入力してサインアップ

### 2. 新しいフォームの作成

1. ダッシュボードで「+ New Form」をクリック
2. フォーム名を入力（例：「User Value Inc. Contact Form」）
3. 通知を受け取りたいメールアドレスを入力
4. 「Create Form」をクリック

### 3. フォームIDの取得

1. 作成したフォームをクリック
2. 「Integration」タブを選択
3. エンドポイントURLをコピー（例：`https://formspree.io/f/xyzabc123`）

### 4. コードへの適用

`pages/Contact.tsx`ファイルの30行目付近にある以下の行を編集します：

```typescript
// 変更前
const FORMSPREE_ENDPOINT = 'https://formspree.io/f/YOUR_FORM_ID';

// 変更後（YOUR_FORM_IDを実際のIDに置き換える）
const FORMSPREE_ENDPOINT = 'https://formspree.io/f/xyzabc123';
```

### 5. テスト送信

1. 開発サーバーを起動：`npm run dev`
2. お問い合わせページにアクセス
3. フォームに情報を入力して送信
4. Formspreeのダッシュボードで送信内容を確認
5. 登録したメールアドレスに通知が届くことを確認

## 📧 Formspreeの機能

### 無料プラン
- 月50件までの送信
- スパムフィルタリング
- メール通知
- 基本的なフォーム分析

### 有料プラン（必要に応じて）
- より多くの送信数
- ファイルアップロード
- カスタムドメイン
- 高度な分析

## 🎨 カスタマイズオプション

### メール通知のカスタマイズ

Formspreeのダッシュボードで以下の設定が可能です：

1. **自動返信メール**: 送信者に自動返信を送る
2. **通知先の追加**: 複数のメールアドレスに通知を送る
3. **件名のカスタマイズ**: メール件名をカスタマイズ

### フォームのカスタマイズ

`pages/Contact.tsx`で以下をカスタマイズできます：

- フォームフィールドの追加/削除
- バリデーションルール
- 成功/エラーメッセージ
- デザインとスタイル

## 🔒 セキュリティ

- Formspreeは自動的にスパムフィルタリングを提供
- reCAPTCHAの追加も可能（有料プラン）
- HTTPS通信で安全にデータを送信

## 🛠️ トラブルシューティング

### フォームが送信されない場合

1. **ブラウザのコンソールを確認**
   - F12キーを押して開発者ツールを開く
   - Consoleタブでエラーメッセージを確認

2. **Formspree IDを確認**
   - `FORMSPREE_ENDPOINT`が正しく設定されているか確認

3. **ネットワークタブを確認**
   - Networkタブで送信リクエストのステータスを確認

### メールが届かない場合

1. **迷惑メールフォルダを確認**
2. **Formspreeのダッシュボードで送信履歴を確認**
3. **通知メールアドレスが正しいか確認**

## 📝 代替オプション

Formspree以外にも以下のサービスが利用可能です：

### EmailJS
- 無料プラン: 月200件
- クライアントサイドのみで完結
- セットアップ: [EmailJS](https://www.emailjs.com/)

### Netlify Forms
- Netlifyでホスティングしている場合に利用可能
- 無料プラン: 月100件
- セットアップ: [Netlify Forms](https://www.netlify.com/products/forms/)

### SendGrid
- より高度なメール送信機能
- API経由での送信
- セットアップ: [SendGrid](https://sendgrid.com/)

## 💡 ヒント

- 本番環境にデプロイする前に、必ずテスト送信を行ってください
- Formspreeの無料プランで十分な場合が多いですが、送信数が多い場合は有料プランを検討してください
- 自動返信メールを設定すると、ユーザー体験が向上します
